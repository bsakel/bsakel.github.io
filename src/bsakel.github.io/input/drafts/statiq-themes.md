# Building a Custom Statiq Theme

Statiq makes it easy to create your own theme using Razor templates and CSS.

## Layout Structure

Your _Layout.cshtml is the root template that wraps all content.

## Tips for Theming

- Use CSS custom properties for easy color changes
- Keep layouts minimal and semantic
- Test on mobile devices

A well-crafted theme makes your site stand out.
