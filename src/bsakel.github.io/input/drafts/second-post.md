# Getting Started with Statiq

Statiq.Web is a powerful static site generator for .NET developers. Here's how I set up this blog.

## Why Statiq

As a C# developer, Statiq lets me use familiar tools. Razor templates, NuGet packages, and the .NET ecosystem.

## Basic Setup

Create a new console app, add the Statiq.Web package, and configure the bootstrapper in Program.cs.

## Content Structure

Put your markdown files in the input directory. Statiq handles the rest.
