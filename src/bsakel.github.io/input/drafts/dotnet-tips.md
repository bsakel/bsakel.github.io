# Essential .NET Tips

A collection of practical tips I've gathered over years of .NET development.

## Use Records for DTOs

Records give you value equality, immutability, and concise syntax out of the box.

## Prefer Async All the Way

Don't mix sync and async code. Use async from the top of the call stack to the bottom.

## Leverage Source Generators

Source generators can eliminate boilerplate code at compile time without runtime reflection.
