# AI Tools I Actually Use as an Engineering Manager

There's a lot of noise about AI in software engineering. Here's what I actually find useful in my day-to-day work.

## Code Review Assistance

I use Claude to help me review pull requests when I need to ramp up on unfamiliar parts of the codebase quickly.

## Writing First Drafts

RFCs, post-mortems, sprint summaries. I write bullet points, then use an LLM to generate a first draft. I always rewrite it, but starting from something is faster than starting from nothing.

## What I Don't Use It For

- 1-1 notes -- too personal
- Performance reviews -- the words need to be mine
- Architecture decisions -- needs deep context

AI tools are accelerators, not replacements.
